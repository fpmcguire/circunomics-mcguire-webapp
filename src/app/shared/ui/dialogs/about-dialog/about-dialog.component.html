About this app

What this app does

Circunomics Trending Repos highlights the most-starred repositories created recently on GitHub, using GitHub's public search API. Results are ranked by star count and refresh each time you load the page.

You can browse the list, open a repository for more detail, and leave a personal star rating. No account or login is needed.

Browsing modes

Two ways to browse the list are available. Switch between them using the toggle in the page header.

Paginated
Shows a fixed page of results. Use the Previous and Next controls to move between pages. Good for focused, deliberate browsing.
Infinite scroll
Loads more results automatically as you scroll to the bottom of the list. Good for continuous discovery.

You can also set your preferred mode via the URL: ?mode=paginated or ?mode=infinite

Ratings

Click any repository name to open its details. From there you can assign a 1–5 star rating.

Ratings are saved locally in this browser and are not sent to GitHub or any external service. They remain available on future visits in the same browser unless you clear browser storage.

Data source

All repository data comes from the GitHub public Search API. The app reads publicly available repository information only — it does not write to, modify, or interact with any GitHub repository.

GitHub API limits

GitHub's public API allows a limited number of requests per hour from unauthenticated clients. In periods of heavy use, or if many requests have already been made from your network, the API may temporarily decline requests.

If this happens, the app shows a clear error message and a Try again button that retries the last request immediately. If the limit window is still active, waiting a few minutes before trying again is usually enough to restore normal access.

In a secure development or deployment environment, adding a GitHub personal access token raises the rate limit substantially. No token is required for normal browsing and no credentials are stored in the app's source code.
